# Kovi

Kovi 是一个简单且可拓展的聊天机器人开发框架，如果你想用 Rust 来开发 Milky / OneBotV11 机器人，那么 Kovi 是一个很好的选择。

🎯 目标是 Rust 最简单 Chat Bot 框架！复杂的 Rust 简化写法？ Kovi 已经尽力了。

🤔 让我数数，文档里面的快速上手，居然10行代码就可以实现一个最简插件。

🥁 还有 CLI 工具，方便项目开发。

🖥️ 只想有趣? 或者定制? Kovi 皆可实现

🛍️ 插件商店带来绝佳 Kovi 购物体验，一键接入插件开发者们的包裹📦。

😍 本项目文档非常简单易懂，跟着来一遍保证都会了。

## 协议说明

Kovi crate 只是一个 `事件 Bus` 插件运行器。当然可以自己去适配更多对接服务。

只不过 Kovi 目前仅将目光聚焦在：

- Milky WebSocket 协议
- OneBotV11 正向 WebSocket 协议
